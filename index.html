Ersteinrichtung

Verbinde das Portal mit deiner Supabase-Datenbank. Diese Daten werden sicher im Browser gespeichert.

So findest du deine Supabase-Zugangsdaten:
1. Melde dich auf supabase.com an
2. Öffne dein Projekt → Settings → API
3. Kopiere Project URL und anon public Key

Aus Settings → API → anon public
